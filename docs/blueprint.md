# **App Name**: Persona Portfolio

## Core Features:

- Header Navigation: Implement a responsive header with navigation links to Home, Resume, Work, and Contact pages.
- Resume Section: Design a Resume page with a sidebar including links to Experience, Certifications, Skills, Education, and About Me sections.
- Page Transitions: Incorporate Framer Motion for smooth page transitions.
- Theme Toggle: Develop a theme toggle button in the header for switching between light and dark themes.
- Landing page with image: Design a landing page that features a prominent image.

## Style Guidelines:

- Primary color: A saturated, vibrant indigo (#663399) to convey both professionalism and trendiness, reminiscent of a creative agency's bold branding.
- Background color: A very light grayish-indigo (#F0F0F5) for the light theme and a dark charcoal-indigo (#202025) for the dark theme.
- Accent color: A contrasting violet (#A020F0) to draw attention to interactive elements and important information, ensuring key features stand out.
- Recommended fonts: 'Poppins' (sans-serif) for headlines and shorter sections of text to give a precise, contemporary feel; paired with 'PT Sans' (sans-serif) for longer text to add readability with its modern yet humanistic style.
- Employ a clean and modern layout utilizing Shadcn UI components and Tailwind CSS for a professional and trendy aesthetic. Prioritize responsiveness for compatibility across different screen sizes.
- Incorporate a set of professional, minimalist icons. Each icon should clearly represent its corresponding section or function to enhance usability and visual appeal.
- Implement subtle and smooth animations to enhance user experience. Use framer motion to transition effects between different pages or components.
